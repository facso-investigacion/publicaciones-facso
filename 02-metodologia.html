2  Metodología

2.1 Datos

2.1.1 Publicaciones

La principal fuente de datos para publicaciones fue el mantenedor de publicaciones de SEPA-VID gestionado por la Unidad de Datos e Indicadores de la VID. Se filtraron las publicaciones adscritas a la Facultad de Ciencias Sociales para el período comprendido entre 2020 y 2025. Cabe destacar que SEPA-VID contiene únicamente las publicaciones de un académico durante la vigencia de su nombramiento en la Universidad, por lo que publicaciones previas al ingreso de un académico no se consideran en el análisis. Además, SEPA-VID contiene mayoritariamente artículos indexados en las bases de datos de WoS, Scopus y Scielo.

Esta base fue complementada a través de la API de ORCID para los 72 investigadores incluidos en el Colaboratorio de Investigación en Ciencias Sociales de la Facultad. Este procedimiento resultó particularmente útil para recoger información de libros, capítulos de libros y artículos indexados en bases de datos distintas a las mencionadas.

Además, la base fue enriquecida a través de la Web of Science Master List, la Scopus Source title list y la API de Scielo con el fin de identificar la indexación de revistas sin ese registro. Además, se extrajo de estas listas información sobre el idioma de la revista o si la publicación tiene acceso abierto. Se utilizó la API de Scopus para recoger la cantidad de citas para cada publicación identificada. Por último, se utilizó el SCImago Journal Rank para obtener información sobre impacto y el cuartil en que se posicionan las revistas registradas.

2.1.2 Académicos

Se utilizó la base de personal académico de la Facultad de Ciencias Sociales para extraer información sobre características del cuerpo académico de la Facultad. Se incluyeron en este estudio todos los nombramientos vigentes al 31 de diciembre del 2025, excluyendo a los investigadores postdoctorales. Se utilizó el RUN de los académicos y académicas para cruzar con la base de publicaciones.

2.2 Variables

La variable principal del estudio son las publicaciones producidas por académicos y académicas de la Facultad de Ciencias Sociales entre 2020 y 2025. Se incluyeron en el estudio tres tipos de publicaciones: artículos de revistas, capítulos de libros y libros. Con la información disponible, no fue posible distinguir entre libros monográficos y libros editados, por lo que la etiqueta “libros” aplica a ambos tipos de publicaciones. Se excluyeron del análisis material editorial, correcciones, cartas, revisiones de libros y otros tipos de publicaciones menores. De tal modo, se incluyeron en el análisis un total de 1104 publicaciones diferentes, de las cuales la mayoría corresponden a artículos (900), seguido de 161 capítulos de libros y 43 libros. Dado que una publicación puede tener más de un autor perteneciente a la Facultad, se encontraron un total de 1273 publicaciones-autor.

Se consideraron, además, variables de caracterización de los académicos de la Facultad tales como departamento, género, edad y jerarquía. Para posibilitar la comparación entre estas variables, la mayoría de los análisis se ponderaron por Jornadas Completas Equivalentes (JCE), calculadas como:

\[JCE = \frac{\sum_{i=1}^{n} h_i}{44}\]

Donde:

  • \(h_i\) = horas reales de dedicación académica de la persona \(i\)

  • \(n\)= número total de personas consideradas en el cálculo

  • \(44\)= número de horas semanales definidas como una jornada completa

Se tomó como referencia a los académicos con nombramiento vigente al 31 de diciembre de 2025. En total, se encontraron 148.91 JCE. En la Tabla 2.1, se presenta un resumen desagregado de Jornada Completas Equivalente por departamento, género, edad y jerarquía.

En el caso de los artículos, se utilizaron variables de caracterización tales como:

  • Indexación (WoS, Scopus, Scielo, Otras). En caso de las revistas multiindexadas, se estableció una jerarquía basada en el prestigio, exigencia y ponderación otorgada en los grupos de evaluación Fondecyt: WoS > Scopus > Scielo > Otras. Es decir, si una revista se encuentra indexada tanto en WoS como en Scopus, se consideró exclusivamente como WoS; si se encuentra en Scopus y en Scielo, pero no en WoS, se contabilizó como Scopus. Lo mismo con Scielo respecto a la categoría “Otra”. Esta última categoría incluye publicaciones en el Emerging Sources Citation Index (ESCI), la European Reference Index for the Humanities and Social Sciences (ERIH Plus), Latindex, el Directory of Open Access Journals (DOAJ), entre otras.

  • Número de Citas (Scopus)

  • Cuartil de Revistas (SJR)

  • Acceso abierto

  • Idioma

Dado que la base final se construyó desde diversas fuentes, el tamaño real de los datos analizados puede cambiar para cada variable. En Tabla 2.2 se resume la cantidad de artículos que cuentan con datos válidos para cada variable antes mencionada.

Por último, como medida complementaria para medir el impacto académico de las publicaciones analizadas, se calculó un índice h agregado a nivel de Facultad y departamento, así como por género, edad y jerarquía. Este índice no debe entenderse como la suma o el promedio de los índices h individuales de los académicos de la Facultad. Más bien, indica cuántas publicaciones \(h\), del total analizado, cuentan con \(h\) o más citas.

2.3 Métodos

Se realizó un análisis cuantitativo descriptivo a partir de las variables antes introducidas. Para análisis agregados a nivel de la Facultad se tomaron como referencias las publicaciones únicas identificadas. En cambio, para la caracterización a partir del género, edad y jerarquía de los autores, se utilizó como unidad de análisis cada par de publicación-autor.

Las bases de datos fueron recolectadas, procesadas y analizadas mediante el lenguaje de programación R. Los datos y el código de análisis, incluyendo el que produce este documento, se encuentran alojados en un repositorio de la Dirección de Investigación en GitHub.

En el caso de los artículos, se analizó tanto volumen como cantidad de citas, impacto, indexación, acceso abierto e idioma. En cambio, dada la baja calidad de la información relativa a libros y capítulos de libros, se optó por limitar su análisis al volumen de producción y sus cruces con características de los autores.

Se tomó como referencia a los académicos con nombramiento vigente al 31 de diciembre de 2025. En total, se encontraron 148.91 JCE. En la @tbl-acad, se presenta un resumen desagregado de Jornada Completas Equivalente por departamento, género, edad y jerarquía.

```{r}
#| label: tbl-acad
#| tbl-cap: "Jornadas Completas Equivalente Facultad de Ciencias Sociales"

load("input/acad-proc.rdata")

library(dplyr)
library(kableExtra)

acad <- acad |> filter(jerarquia != "Investigador Postdoctoral")

# Total
jce_totales <- acad |> 
  summarise(grupo = "Total", jce = sum(horas_reales)/44)

# Por departamento
jce_depto <- acad |> 
  group_by(departamento) |> 
  summarise(jce = sum(horas_reales)/44) |> 
  rename(grupo = departamento)

# Por sexo
jce_genero <- acad |> 
  group_by(sexo) |> 
  summarise(jce = sum(horas_reales)/44) |> 
  rename(grupo = sexo)

# Por tramo de edad (corregido: agrupa por edad_tramo, no por edad)
jce_edad <- acad |> 
  mutate(edad_tramo = case_when(
    edad < 50 ~ "Menos de 50 años",
    edad >= 50 & edad < 60 ~ "50 a 59 años",
    edad >= 60 ~ "60 años o más"),
    edad_tramo = factor(edad_tramo, 
      levels = c("Menos de 50 años", "50 a 59 años", "60 años o más"))) |> 
  group_by(edad_tramo) |> 
  summarise(jce = sum(horas_reales)/44) |> 
  rename(grupo = edad_tramo)

# Por jerarquía
jce_jerarquia <- acad |> 
  filter(jerarquia != "Investigador Postdoctoral") |> 
  group_by(jerarquia) |> 
  summarise(jce = sum(horas_reales)/44) |> 
  rename(grupo = jerarquia)

# Combinar todo en una sola tabla
tabla_jce <- bind_rows(
  jce_totales,
  jce_depto,
  jce_genero,
  jce_edad,
  jce_jerarquia
)

# Construir la tabla con kbl y pack_rows
tabla_jce |> 
  mutate(jce = round(jce, 2)) |> 
  kbl(
    col.names = c("Grupo", "JCE"),
    align = c("l", "c")) |> 
  kable_styling(full_width = FALSE, bootstrap_options = c("striped", "hover")) |> 
  pack_rows("Total", 1, 1) |> 
  pack_rows("Departamento", 2, 1 + nrow(jce_depto)) |> 
  pack_rows("Sexo", 2 + nrow(jce_depto), 1 + nrow(jce_depto) + nrow(jce_genero)) |> 
  pack_rows("Tramo de edad", 
            2 + nrow(jce_depto) + nrow(jce_genero), 
            1 + nrow(jce_depto) + nrow(jce_genero) + nrow(jce_edad)) |> 
  pack_rows("Jerarquía", 
            2 + nrow(jce_depto) + nrow(jce_genero) + nrow(jce_edad), 
            nrow(tabla_jce))

```

# Tabla de conteos no-NA por variable
tabla_conteos <- tibble(
  variable = c("Indexación", "Cuartil", "N° de citas", "Acceso abierto (OA)", "Idioma"),
